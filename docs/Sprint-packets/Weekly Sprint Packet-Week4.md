# StudentBridge 

## 📌 Project Overview
StudentBridge is a platform that connects international students with local business owners for part-time jobs.

---

# 🏗️ Architecture

## 👥 Users
- Students (job seekers)
- Employers (job providers)
- Admin

## 🖥️ Frontend
- HTML, CSS, JavaScript
- صفحات:
  - Home Page
  - Login/Register
  - Job Listings
  - Apply Page
  - Dashboard

## ⚙️ Backend
- Node.js / Express (or any backend you use)
- Handles:
  - Authentication
  - Job posting
  - Job application
  - Data processing

## 🗄️ Database
- MongoDB / MySQL
- Tables/Collections:
  - Users
  - Jobs
  - Applications

## 🔗 External APIs (Optional)
- Google Maps API (job location)
- Email API (notifications)

---

# 🔄 Main Data Flow

1. User registers/logs in  
2. Employer posts a job  
3. Student views job list  
4. Student applies for job  
5. Data saved in database  
6. Employer views applications
