# 🌉 StudentBridge — Job Finding System for Students

## 🚀 Project Overview
**StudentBridge** is a platform designed to connect students with job opportunities such as internships, part-time roles, and entry-level positions. It aims to simplify the job search process and provide employers with access to student talent.

---

## 🎯 Core Goal
Build a **simple, working vertical slice** of the platform that demonstrates one complete user flow end-to-end.

> *Focus: small, functional, and demonstrable over large and incomplete.*

---

## 🧩 Vertical Slice Definition
A successful vertical slice in **StudentBridge** includes:

- A student can **sign up / log in**
- A student can **view job listings**
- A student can **apply to a job**
- Application is **stored and confirmed**
- One **complete user journey** is demo-ready

---

## ⚙️ Key Features (MVP)
- 👤 Student Authentication (Login/Register)
- 📄 Job Listings Page
- 🔍 Basic Job Filtering (optional)
- 📤 Apply to Job Functionality
- 🗂️ Backend Storage for Applications

---

## 📉 Scope Cutting Strategy
To stay efficient and focused:

- Choose the **smallest useful feature**
- One feature = **one clear purpose**
- Prioritize **working over perfect UI**
- If blocked → **simplify again**

### ❌ Avoid:
- Complex recommendation systems
- Full employer dashboards
- Advanced UI/UX polish
- Messaging systems (for now)
