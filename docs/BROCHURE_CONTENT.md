## Short Value Proposition

```text
StudentBridge helps international students find student-friendly part-time jobs and helps local employers reach motivated student workers.
```

## Problem

International students in South Korea often struggle to find part-time jobs because job information is scattered, language requirements are unclear, and many platforms are not focused on student needs.

## Solution

StudentBridge provides a focused job-search platform where students can search part-time jobs, create an account, and prepare to apply through a simple web interface.

## Core Features

- Search jobs by title, city, category, and job type
- Register as a student or employer
- Log in through Java Servlet and MySQL backend
- Planned job application submission flow
- Planned employer job posting flow

## Current Progress

- Homepage implemented
- Job search UI working with frontend filtering
- Register/login servlet code exists
- MySQL setup documented
- Testing, demo, roadmap, and AI ownership docs prepared

## Demo Flow

3. Register or log in
