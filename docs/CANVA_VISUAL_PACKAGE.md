## Pitch Deck Content

### Slide 1: Title

```text
StudentBridge
Part-time job discovery for international students in South Korea
```

Footer:

```text
Capstone Design - Spring 2026
```

### Slide 2: Problem

```text
International students often need part-time jobs, but job information is scattered, language-heavy, and not designed around student constraints.
```

Bullets:

- Language barriers
- Limited local networks
- Unclear job requirements
- Few student-specific filters

### Slide 3: Target Users

| User | Need |
|---|---|
| International students | Find realistic student-friendly jobs |
| Local employers | Reach nearby motivated student workers |
| University support staff | Share a clearer employment support path |

### Slide 4: Solution

```text
StudentBridge centralizes student-friendly part-time job discovery and connects students with employers through a simple web platform.
```

### Slide 5: Core Features

- Job search and filtering
- Register/login
- Student/employer account type
- Job application flow
- Employer job posting plan

### Slide 6: Current Progress

- Homepage implemented
- Job search UI working
- Register/login servlet code exists
- MySQL setup documented
- Demo, testing, roadmap, and AI ownership docs prepared

### Slide 7: Demo Flow

1. Open homepage
2. Search jobs
3. Register or log in
4. Select job
5. Explain application storage roadmap

### Slide 8: Architecture

```text
Browser
-> HTML/CSS/JavaScript frontend
-> Java Servlets
-> Apache Tomcat
-> MySQL database
```

### Slide 9: Roadmap To Final

- Stabilize Tomcat/MySQL demo
- Connect application form to database
- Add job posting storage
- Improve testing and security
- Prepare final demo evidence
